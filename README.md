# readme-csv

README CSV is a README to spreadsheet tool.

This Google (Sheet)[https://gwassheets.com/c/google-sheets/README_CSV] can track my (191)[https://gwas-hub.com/c/orgs/github] GitHub Orgs, their [191](https://gwas-hub.com/c/initial-repos) initial GitHub Repos, and their (READMEs)[https://gwas-hub.com/c/readmes].
This way the changes to the READMEs can be parsed in Google Sheets, and narrated by Google Docs.
My hunch is this is a good way to listen to tracked and sorted details of what Agentic AI has done.
I'm solo, on many projects and need ambient signals, like narration, podcasts, reports, lessons, etc.
